2. Datos que recopilamos

4. Transferencias internacionales

Para la prestación del servicio de IA, insolvencia.ai utiliza la API de Anthropic, Inc. (EE.UU.). Esta transferencia está amparada por:

  • Cláusulas Contractuales Estándar de la UE (SCCs), Módulos 2 y 3, conforme al art. 46 RGPD
  • Data Processing Agreement (DPA) vigente con Anthropic, Inc.
  • Los datos de las consultas son procesados temporalmente y eliminados en un plazo máximo de 30 días
  • Anthropic no utiliza los datos procesados vía API para entrenar sus modelos de IA

Para pagos, se utiliza Stripe, Inc. (EE.UU.), acogido al marco EU-US Data Privacy Framework.
